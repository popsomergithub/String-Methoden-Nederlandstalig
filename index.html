'String length'


De lengte van een string wordt bepaald door het aantal UTF-16-code-eenheden te tellen die deze bevat.

Dit betekent dat de eigenschap length u vertelt uit hoeveel 16-bits eenheden de tekenreeks bestaat, en niet noodzakelijkerwijs uit het aantal tekens dat erin wordt weergegeven.

Bij normale karakters is het geen probleem (die zijn allemaal 1 karakter lang), maar bij emojis kan dat variëren van 1 to 3 karakters lang. (Je moet dat dus incalculeren)



Enkele voorbeelden
Resultaten
"abcdefghijklmnopqrstuvwxyz".length
26
"".length
0
"*****1*****2".length
12
"-+!,;:=@&é'".length
11
"John Doe".length
8

'String charAt()'


Met de `charAt()`-methode kun je een specifiek teken uit een string halen door de positie (index) van het karakter binnen de string op te geven.

Werkt alleen op strings: het is specifiek ontworpen voor stringobjecten. (in tegenstelling tot de At() methode)

Accepteert alleen positieve indices: u kunt geen negatieve indices gebruiken om toegang te krijgen tot tekens vanaf het einde van de tekenreeks.

Er wordt een fout gegenereerd voor ongeldige indices: als u probeert toegang te krijgen tot een index die groter is dan de lengte van de tekenreeks, wordt er een IndexError gegenereerd.


Enkele voorbeelden
Resultaten
"Abra CaDaBra".charAt(6)
a
"Abra CaDaBra".charAt(0)
A
"Abra CaDaBra".charAt(7)
D
"Abra CaDaBra".charAt(11)
a
"Abra CaDaBra".charAt(9)
B

'String char-code-at()'


Met de methode charCodeAt() kunt u de numerieke code vinden die een specifiek teken in een tekenreeks vertegenwoordigt.

Je geeft de positie (index) in de string, en het geeft je een getal tussen 0 en 65535, wat de code van het karakter is volgens de UTF-16-standaard.


Enkele voorbeelden
Resultaten
"Hallo Iedereen".charCodeAt(6)

"Hallo Iedereen".charCodeAt(0)
72 (H)
"Hallo Iedereen".charCodeAt(7)
101 (e)
"Hallo Iedereen".charCodeAt(13)
110 (n)
"Hallo Iedereen".charCodeAt(8)
100 (d)

'String at()'


De at()-methode is een geheel nieuwe functie in JavaScript. Het is een handige manier om tekens uit een string te halen.

Wat deze methode anders maakt dan de oudere charAt()-methode, is dat je met at() negatieve getallen kunt gebruiken om tekens vanaf het einde van de tekenreeks te tellen.

Stel dat u bijvoorbeeld een tekenreeks hebt met de naam myString. Als je vroeger het op een na laatste teken wilde krijgen met charAt(), moest je wat rekenwerk doen: charAt(myString.length - 2).

Met at() kun je nu gewoon myString.at(-2) schrijven en hetzelfde resultaat krijgen, wat veel eenvoudiger is!


Enkele voorbeelden
Resultaten
"Dit is een test".charAt(-2)
lege string
"Dit is een test".at(-2)
s
"Dit is een test".at(7)
E

'String []'


Het gebruik van vierkante haken om toegang te krijgen tot tekens in een tekenreeks kan een beetje verwarrend zijn:

Het lijkt erop dat je met een array te maken hebt, maar strings zijn eigenlijk anders.

Als je toegang probeert te krijgen tot een teken dat niet bestaat, krijg je door vierkante haakjes te gebruiken een waarde van 'ongedefinieerd' terug, terwijl de methode charAt() je een lege string geeft.

U kunt de tekens in een tekenreeks niet wijzigen met vierkante haakjes. Ook al ziet u mogelijk geen foutmelding, een poging om str[0] = "A" uit te voeren, zal de tekenreeks niet daadwerkelijk veranderen.


Enkele voorbeelden
Resultaten
"Niemand is Perfect"[0]
N
"Niemand is Perfect"[0] = "A"
Niemand is Perfect
"Niemand is Perfect"[11]
P

'String slice()'


Je kunt op drie verschillende manieren een stuk tekst uit een string halen:

slice(start, end)

substring(start, end)

substr(start, length)

Met de slice()-methode kun je een deel van een string nemen en met alleen dat deel een nieuwe string maken.
Je vertelt waar het moet beginnen en waar het moet eindigen (maar de eindpositie is niet opgenomen in de nieuwe string).


Enkele voorbeelden
Resultaten
"Aardbei, Citroen, Framboos".slice(9, 16)
Citroen
"Aardbei, Citroen, Framboos".slice(18,26)
Framboos
"Aardbei, Citroen, Framboos".slice(0, 7)
Aardbei
"Aardbei, Citroen, Framboos".slice(22, 26)
boos
"Aardbei, Citroen, Framboos".slice(19,22)
ram

'String substring()'


De methode substring() lijkt sterk op slice().
Het belangrijkste verschil is dat als je negatieve getallen probeert te gebruiken voor de begin- of eindpositie, substring() deze als 0 behandelt, terwijl slice() ze anders behandelt.


Enkele voorbeelden
Resultaten
"Prei, Wortel, Raap".slice(-2, 6)
empty string
"Prei, Wortel, Raap".substring(-2, 6)
Prei,

'String concat()'


De methode concat() combineert twee of meer strings in één enkele, langere string.

TIP: als je spatie wil tussen je teksten moet je wel een spatie intikken aan het einde van uw vorige tekst, of aan het begin van de volgende tekst...


Voorbeeld
Resultaten
"Peter".concat("Opsomer")
PeterOpsomer
"Kat, Paard, ".concat("Hond")
Kat, Paard, Hond

'String replace()'


Met de methode replace() kun je een specifiek deel van een string vinden en dit vervangen door iets anders.

Alleen het eerste deeltje tekst dat gelijk is aan de zoektekst wordt vervangen. Komen er nog zulke stukjes tekst voor, dan worden die niet vervangen.


Enkele voorbeelden
Resultaten
"blablabla".replace("la", "xx")
bxxblabla
"String Kopiëren".replace("Kopië", "Duplice" )
String Dupliceren

'String replaceAll()'


JavaScript heeft in 2021 een nieuwe stringmethode gekregen, genaamd replaceAll(). Deze methode is vergelijkbaar met replace(), maar laat je een reguliere expressie gebruiken (waarbij de vlag 'g' is ingesteld) om meerdere exemplaren van een patroon binnen een string te vinden en te vervangen. Als u geen reguliere expressie met de 'g'-vlag gebruikt, krijgt u een foutmelding.


Enkele voorbeelden
Resultaten
"blablabla".replace("la", "xx")
bxxblabla
"String Kopiëren".replace("Kopië", "Duplice" )
String Dupliceren

'String split()'


Je kunt een string in een array van kleinere stukjes veranderen met behulp van de split()-methode. Als je niet vertelt hoe de string moet worden gesplitst (dat wil zeggen: je geeft er geen scheidingsteken aan), dan zal de hele string het enige item in de array zijn. Als u echter zegt dat het moet splitsen met een lege tekenreeks (""), wordt de tekenreeks opgesplitst in een reeks afzonderlijke tekens.


Enkele voorbeelden
Resultaten
"abcdabc".split('d')
[ 'abc', 'abc' ]
"Muis, Kikker, Hond, Ezel".split(",")
[ 'Muis', ' Kikker', ' Hond', ' Ezel' ]
